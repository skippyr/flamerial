# flamerial
An eccentric 8-bits flameish theme.

![](preview.png)

## Install
In order to install it, follow the theming instructions of your apps to use
these theme files.

## Palette
These are the colors used in it ordered by their ANSI value:

| ANSI | Name    | Hex       |
| `0`  | Black   | `#150d06` |
| `1`  | Red     | `#b31c13` |
| `2`  | Green   | `#6c7422` |
| `3`  | Yellow  | `#c76800` |
| `4`  | Blue    | `#b08548` |
| `5`  | Magenta | `#8a3820` |
| `6`  | Cyan    | `#9c3900` |
| `7`  | White   | `#e9d397` |

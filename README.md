# flamerial
An 8-bits terminal dark theme with flameish colors.

![](preview.png)

## Install
In order to install it, follow the theming instructions of your apps to use
these theme files.

## Palette
These are the colors used in it ordered by their ANSI value:

```
#150d06
#b31c13
#990101
#c76800
#b08548
#8a3820
#9c3900
#e9d397
```

## ❡ Install

&emsp;**1.**&ensp;Copy the theme file flamerial.colorscheme to ~/.local/share/konsole:

```sh
mkdir -p ~/.local/share/konsole;
cp flamerial.colorscheme ~/.local/share/konsole;
```

&emsp;**2.**&ensp;Apply the theme in your konsole profile.

## ❡ Install

- Copy the theme file flamerial.colorscheme to ~/.local/share/konsole:

```sh
mkdir -p ~/.local/share/konsole;
cp flamerial.colorscheme ~/.local/share/konsole;
```

- Apply the theme in your konsole profile.

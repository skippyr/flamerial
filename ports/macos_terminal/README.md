## ❡ Install
### Procedures
- Open the macOS Terminal settings.
- Select `Profile > Text > ... > Import...`.
- Open `Flamerial.terminal`.
- Select the `Flamerial` theme and click on `Default`.
- Restart the macOS Terminal.

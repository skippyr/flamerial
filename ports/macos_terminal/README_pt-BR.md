## ❡ Instalação
### Procedimentos
- Abra as configurações do macOS Terminal.
- Selecione `Profile > Text > ... > Import...`.
- Abra `Flamerial.terminal`.
- Selecione o tema `Flamerial` e clique em `Default`.
- Reinicie o macOS Terminal.

## ❡ Install
### Procedures
- Copy the file `flamerial.sh` to `~/.config/bspwm`:

```zsh
mkdir -p ~/.config/bspwm;
cp flamerial.sh ~/.config/bspwm;
```

- Add the following line to `~/.config/bspwm/bspwmrc`:

```zsh
~/.config/bspwm/flamerial.sh &
```

- Reopen bspwm.

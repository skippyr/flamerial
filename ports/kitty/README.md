## ❡ Install
- Copy `flamerial.conf` to `~/.config/kitty/themes`:

```zsh
mkdir -p ~/.config/kitty/themes;
cp flamerial.conf ~/.config/kitty/themes;
```

- Include it in `~/.config/kitty/kitty.conf`:

```zsh
include themes/flamerial.conf
```

- Reload Kitty.

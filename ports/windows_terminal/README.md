## ❡ Install
### Procedures
- Open the Windows Terminal settings.
- Select `Open JSON file`.
- Add the object defined in `flamerial.json` to the `schemes` array.
- Select `Profiles > Defaults`, then, under `Appearance`, change the color scheme to `Flamerial`.
- Click on `Save`.

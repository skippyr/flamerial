## ❡ Instalação
### Procedimentos
- Copie `flamerial` para `~/.config/ghostty/themes`:

```zsh
mkdir -p ~/.config/ghostty/themes;
cp flamerial ~/.config/ghostty/themes;
```

- Use ele em `~/.config/ghostty/config`:

```zsh
theme = flamerial
```

- Reinicie o Ghostty.
